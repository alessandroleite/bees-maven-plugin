The CloudBees Plugin for Maven allows you to deploy WAR-based projects to CloudBees.


Sample Usage
------------
    mvn bees:run

    mvn bees:deploy

[Full Documentation](http://wiki.cloudbees.com/bin/view/RUN/MavenGuide)
